# Expert Wine Advisor

A premium, mobile-responsive wine consulting website built with Next.js and Tailwind CSS. The site positions Andrew Misialek, a wine manufacturing professional who studied viticulture at UC Davis, as a credible advisor for wine brands, restaurants, hospitality groups, private collectors, and events.

## Tech stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Vercel-ready deployment

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
```

## Deploy to Vercel

```bash
vercel --prod
```

Or connect the GitHub repository to Vercel and deploy from the dashboard.

## Placeholders to replace

Before using this as the final business website, replace:

- `hello@example.com` with the business email
- `(555) 555-5555` with the business phone number
- Any exact credentials, schools, certifications, awards, press mentions, or photos once verified

No specific schools, certifications, awards, or unverifiable claims are included in the current copy.
